T-Библиотека
Программа написана на Python

Возможности

Добавление книг: требуется ввести название, автора, жанр, год выпуска, описание к книге.
Просмотр книг:  для просмотра добавленных книг необходимо выбрать в каком порядке и какие книги показать, т.е. можно отсортировать их по названию(по алфавиту), автору(по алфавиту), году(от меньшего к большему); также можно использовать фильтры, чтобы показывались только книги, которые нужны пользователю, например, только романы: для этого ему нужно выбрать фильтр по жанру и вписать нужный жанр в строку.
Добавление в избранное: можно добавлять/удалять книги из избранного по желанию, для этого нужно выбрать пункт №4(добавить/удалить книгу из избранного), для того, чтобы это сделать, необходимо вписать в строку ID книги, все книги и их ID будут указаны выше.
Изменение статуса чтения: Выбрав пункт №5 можно изменить статус книги прочитана/не прочитана. Для этого необходимо выбрать ID книги и ввести новый статус книги(1-прочитана, 2-не прочитана). После этого в списке книг она будет отображаться как прочитанная/не прочитанная.
Возможность удаления книг: выбрав возможность №6, а затем написав ID книги, её можно навсегда удалить из списка. Перед удалением программа попросит подтверждение удаления вводом да/нет.
Поиск нужной киниги: выбрав пункт №7, можно начать поиск по книгам по ключевому слову. Для этого программа попросит ввести ключевое слово для поиска этого слова в названии, авторе книги и описании, после программа выведет все подходящие варианты

Как программа работает?
Функция load_books() проверяет наличие в файле json уже записанных данных, т.е. существующих книг, если такого файла нет, то она создаёт новый список.
Функция save_books(books) позволяет записывать книги в файл json, т.е. позволяет добавлять новые книги
Функция add_book(books) непосредственно запрашивает у пользователя все данные для добавления книги, проверяя что название не пустое, а год - число( если не число, то цикл while пока не введут число). В созданном словаре book программа записывает id книги, который равен длине списка+1,  статус книги, который по умолчанию "не прочитана", избранное (по умолчанию не в избранном).
Функция view_books(books) позволяет просматривать книги, если книг нет, то об этом уведомляется пользователь. в зависимости от выбора порядка показа книг(т.е. ввода "1" и т.д.), идет сортировка книг в зависимости от числа.
Функция find_book_by_id позволяет найти книгу по id
Функция toggle_favourite(books) позволяет пользователю менять значение not book ["favorite"] с True на False и наоборот, предварительно запрашивая у пользователя id книги.
Функция change_status(books) позволяет пользователю поменять значение в поле status с непрочитанной на прочитанную и наоборот.
Функция delete_book(books) позволяет удалить книгу, при этом пересчитывая id оставшихся книг
Функция search_books(books) ищет совпадения с введённым словомв полях название, автор, описание. 
Функция show_favorites(books) создает список из книг, где favorite=True
Функция show_menu() печатает главное меню пользователя. 
Функция main() приветствует пользователя, загружает книги из файла json, показывает меню и ждёт выбора. В зависимости от выбора цифры, начианет работу нужная функция. При выборе опции №8 цикл работы прерывается, идёт сохранение полученных данных.
